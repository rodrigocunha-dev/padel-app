# Como entrar no app sem senha

**Resposta curta:** digite seu número de WhatsApp e você recebe um código de 6 números.
Digite o código e pronto — não existe senha para criar nem para esquecer.

## Passo a passo

1. Abra o app e toque em **"Entrar"**.
2. Digite seu número com DDD. O app formata sozinho enquanto você digita.
3. Toque **"Receber código"**.
4. Abra a mensagem que chegou e veja o código de 6 números.
5. Digite o código e toque **"Confirmar e entrar"**.

## E se o código não chegar?

Toque **"Trocar número ou pedir novo código"** na tela do código. Você volta para a tela anterior e pode conferir se o número está certo antes de pedir de novo.

O código vale por poucos minutos. Se demorar para digitar, peça um novo.

## Privacidade

Seu número serve para entrar e para o clube falar com você sobre um jogo marcado. Ele não aparece para outros jogadores.
